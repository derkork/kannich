# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.9.0] - 2026-03-20
### Changed
- Bumped the `kannich-parent` dependency to `0.9.0`.
- Removed unnecessary dependencies slowing down module loading.

## [0.8.0] - 2026-03-06
### Changed
- `Trivy` now supports the standardized `Tool` execution interface.

## [0.7.0] - 2026-02-25
### Changed
- Update parent pom version.

## [0.6.0] - 2026-02-20
### Added
- Added support for `Tool` interface of Kannich.
- Added a new `scanFs` method to scan a directory and produce a HTML report.

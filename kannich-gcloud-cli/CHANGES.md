# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.4.0] - 2026-03-20
### Changed
- Bumped the `kannich-parent` dependency to `0.9.0`.
- Removed unnecessary dependencies slowing down module loading.

## [0.3.0] - 2026-03-06
### Added
- `GcloudCli` now supports the standardized `Tool` execution interface.

## [0.2.0] - 2026-03-03
### Added
- The `exec` command now returns an `ExecResult` which is useful to parse the output for further processing.

## [0.1.0] - 2026-03-03
### Added
- Initial version.

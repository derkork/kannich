# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.3.0] - 2026-03-04
### Added
- The `exec` command now accepts a `silent` flag which prevents the output from appearing in the Kannich logs.

## [0.2.0] - 2026-03-03
### Added
- The `exec` command now returns an `ExecResult` which is useful to parse the output for further processing.

## [0.1.0] - 2026-03-03
### Added
- Initial version.
